# Installation
Easy to install it via NuGet:

- **Security**

```shell
PM > Install-Package Arcus.WebApi.Security
```

- **Logging**

```shell
PM > Install-Package Arcus.WebApi.Logging
```

For a more thorough overview, we recommend reading our [documentation](#documentation).
